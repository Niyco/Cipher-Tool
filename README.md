# Cipher-Tool
A python program containing tools for encrypting and decrypting classical ciphers

## Ciphers supported:
* Morse Code
* Baconian
* Binary
* Baudot
## Analysis tools:
* Length
* Frequency Analysis
* Index of Coincidence
## Text manipulation tools:
* Uppercase
* Lowercase
* Reverse
* Strip Text
* Block Text
* Add Spaces
